# Pricing — CloudFreedom Agent Platform

Pricing is designed to be simple and aligned with what can be reliably operated on a single Coolify host.

All prices exclude VAT.

## Starter — €49 / month

For a single team or small project.

- Up to **2** agent stacks
- Resource limit per stack: **0.5 vCPU / 512 MB RAM**
- 1 custom domain (CNAME to CloudFreedom)
- Email support (best effort)

## Team — €199 / month

For agencies and product teams.

- Up to **10** agent stacks
- Resource limit per stack: **1 vCPU / 1 GB RAM**
- 5 custom domains
- Basic monitoring + incident notifications
- Business-hours support

## Business — €799 / month

For production workloads.

- Up to **25** agent stacks
- Resource limit per stack: **2 vCPU / 4 GB RAM**
- 20 custom domains
- Backups for stateful stacks (daily)
- Priority support + onboarding session

## Sovereign — €3,000+ / month

For regulated environments and dedicated capacity.

- Custom stack count and resource envelopes
- Optional **dedicated host** or isolated Docker destination
- Custom data retention and compliance requirements
- SLA options

## Add-ons (optional)

- Additional stack packs (e.g. +10 stacks): priced per plan
- Managed template customization
- Dedicated IP / private connectivity (if available)

## Notes

- LLM token usage can be billed separately via CloudFreedom Router metering.
- The MVP focuses on predictable infrastructure quotas rather than complex per-second billing.
